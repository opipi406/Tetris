# Tetris
> #### v1.0.0  

---
## 概要
簡易テトリススクリプト

#### 操作方法
[→]： 右移動  
[←]： 左移動  
[↓]： ソフトドロップ  
[↑]： ハードドロップ  
[Z]： 左回転  
[X]： 右回転  

## デモ
#### 実行
```
python main.py
```
